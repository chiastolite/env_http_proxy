# Introduction

env_http_proxy is a utility for Net::HTTP. When ENV['http_proxy'] is defined, Net::HTTP use it for http_proxy.

## Installation

Add this line to your application's Gemfile:

    gem 'env_http_proxy'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install env_http_proxy
